# 🎰 Slot Machine – Python CLI Game

Slot Machine is a command-line based Python game that simulates a casino-style slot machine.  
Users can deposit money, place bets on multiple lines, and spin the slot machine to try their luck.

---

## 🚀 Features

- 💰 Deposit system with balance tracking  
- 🎯 Betting on multiple lines (up to 3)  
- 🎰 Random slot generation using Python  
- 🧮 Win/loss calculation based on symbol matching  
- 📊 Dynamic balance updates after each spin  
- 🖥️ Interactive command-line interface  

---

## 🛠️ Tech Stack

- Python  
- Built-in `random` module  

---

## 🧠 How It Works

1. User deposits an initial amount  
2. Selects number of lines to bet on (1–3)  
3. Enters bet amount per line  
4. Slot machine generates random symbols  
5. If symbols match across a line → user wins  
6. Balance updates based on result  

---

## 🎲 Game Logic

- The slot machine is a 3x3 grid  
- Symbols have different frequencies and values:

### Symbol Count (Probability)
- A → 2  
- B → 4  
- C → 6  
- D → 8  

### Symbol Value (Payout)
- A → 5x  
- B → 4x  
- C → 3x  
- D → 2x  

Higher value symbols are rarer, increasing game difficulty and realism.
